Locus is a computer vision project designed to automate and streamline the attendance tracking process using facial recognition. It leverages deep learning models to accurately identify individuals in real-time, mapping them to a database to securely and efficiently log their presence.

### Key Technologies
* **Domain:** Computer Vision & Deep Learning
* **Language:** Python
* **Core Functionality:** Real-time face detection, feature extraction, and identity matching.
